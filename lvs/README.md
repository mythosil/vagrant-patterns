Network diagram ( *NAT mode* )
```
                +----------+
                | *client* |
                +-----+----+
                      |
        +-------------+------------+
        |                          |
+-------+--------+        +--------+-------+
| 192.168.30.101 |        | 192.168.30.101 |
| 192.168.30.2   |        | 192.168.30.3   |
+----------------+  VRRP  +----------------+
|  *LVS master*  |<------>|  *LVS backup*  |
+----------------+        +----------------+
| 192.168.31.2   |        | 192.168.31.3   |
| 192.168.31.101 |        | 192.168.31.101 |
+-------+--------+        |--------+-------+
        |                          |
        +--------------------------+
        |                          |
+-------+--------+        +--------+-------+
|  192.168.31.5  |        |  192.168.31.6  |
+----------------+        +----------------+
|    *web1*      |        |     *web2*     |
+----------------+        +----------------+
```

Start machines.
```
$ vagrant up
```

HTTP request.
```
$ curl http://192.168.30.101
<h1>Hello World</h1>
```
